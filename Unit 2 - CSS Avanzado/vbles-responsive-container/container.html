Ejemplo de @container de CSS

Observa cómo el mismo componente .card se ve diferente en el contenedor azul (ancho) y en el rojo (estrecho).

Contenedor Ancho (700px)

Diseño Horizontal

Este contenedor mide más de 400px, por lo que la @container query se activa y la tarjeta cambia a un diseño horizontal.

Contenedor Estrecho (350px)

Diseño Vertical

Este contenedor mide menos de 400px. La @container query no se activa, así que la tarjeta mantiene su diseño vertical por defecto.
